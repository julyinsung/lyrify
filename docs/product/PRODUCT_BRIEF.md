## 1. Product Goal

| 항목 | 내용 |
| --- | --- |
| 목표 | AI 음악 도구(ACE-Step-1.5 등)의 창작 재료(프롬프트, 가사, 스타일)를 자동 감지 및 수집하고, 이를 구글 릴리아(Lelia/Lyrify)에서 생성한 고품질 완성 음원(MP3)과 매핑하여, 어울리는 비주얼(생성형 이미지/템플릿)을 결합해 유튜브 배포용 동영상(MP4) 또는 통합 리소스를 제작/수출해주는 크리에이터 파이프라인 웹 애플리케이션 구축 |
| 주요 사용자 | AI 음악을 창작하고 이를 유튜브/사운드클라우드 등의 소셜 플랫폼에 배포하려는 크리에이터 |
| 해결하려는 문제 | 1. ACE-Step-1.5의 초안(가사/스타일)과 실제 배포용 고품질 음원(구글 릴리아 MP3)이 서로 분리되어 매핑 관리가 어려움.<br>2. 음원에 이미지를 입히고 유튜브 배포용 가사 비디오를 렌더링하는 작업이 번거롭고 반복적임. |
| 성공 기준 | 1. ACE-Step-1.5 디렉토리 감시를 통한 음악 재료(가사/스타일 프롬프트)의 자동 카드화.<br>2. 대시보드에서 음악 재료 카드와 구글 릴리아 음원(MP3) 간의 매핑 기능 제공.<br>3. 가사/스타일에 어울리는 AI 이미지 생성(생성형 API 활용) 또는 커스텀 썸네일 합성 기능.<br>4. [음악 재료 + 매핑된 음원 + 썸네일 이미지]를 결합하여 유튜브용 MP4 동영상 파일로 인코딩 및 로컬 저장 성공. |
| 비목표 | 1. 클라우드 기반 데이터베이스 및 멀티유저 가입 기능 (초기 MVP는 완벽한 로컬 싱글 유저 환경으로 제한).<br>2. 유튜브/사운드클라우드 직접 업로드 자동화 (1단계 MVP에서는 인코딩 완료된 MP4 파일의 로컬 생성 및 내보내기만 포함).<br>3. 구글 릴리아 웹 자동 다운로드 (릴리아 생성물은 사용자가 로컬 감시 디렉토리에 직접 복사/저장하여 통합하는 방식). |

| --- | --- | --- | --- | --- |

| SCN-002 | 구글 릴리아 음원 매핑 | 대시보드에서 음악 재료 카드와 구글 릴리아에서 다운로드한 최종 MP3 파일을 손쉽게 매핑하여 관리할 수 있음 | Must | REQ-001 |

## 3. Release Scope

| 구분 | 내용 |
| --- | --- |
| 이번 릴리즈 포함 (MVP) | 1. 지정 로컬 디렉토리 감시 및 가사/스타일 프롬프트 파싱 기능.<br>2. 음악 재료 정보 시각화(카드 뷰) 및 구글 릴리아 음원(MP3) 수동/규칙 매핑.<br>3. 가사 스타일에 적합한 생성형 이미지(이미지 생성 API 또는 기본 합성 템플릿) 결합 기능.<br>4. 음원과 썸네일 이미지를 합성하여 MP4 동영상 파일로 인코딩해주는 로컬 비디오 내보내기 기능. |
| 이번 릴리즈 제외 | 1. 유튜브/사운드클라우드 API 자동 업로드 연동.<br>2. 구글 릴리아 웹 다운로드 자동 크롤러. |
| 다음 릴리즈 후보 | 1. 유튜브 API 연동 직접 업로드 및 진행률 표시.<br>2. 구글 릴리아 웹 자동 다운로더. |

## 4. Product Risks And Assumptions

| ID | 유형 | 내용 | 대응 |
| --- | --- | --- | --- |
| RISK-001 | Risk | ACE-Step-1.5 결과물 파일 명명 규칙이나 폴더 구조가 달라질 경우 파싱 오류 발생 가능성 | 애플리케이션 자체 설정 파일(`config.json` 등) 또는 웹 UI 내 설정 옵션을 통해 감시 폴더 경로 및 파일 파싱 규칙을 유연하게 조정할 수 있게 설계 |
| RISK-002 | Assumption | 로컬 동영상(MP4) 인코딩을 위해 FFmpeg 라이브러리 및 환경 변수 등록이 필요함 | 서버 기동 시 로컬 FFmpeg 설치 여부를 유효성 검사하고, 미설치 시 대시보드 내에 친절한 설치 가이드를 제공함 |
| RISK-003 | Risk | 이미지 생성 API(예: Google Imagen 등) 연동 실패 시 비주얼 제작 차단 가능성 | API 키가 없거나 실패하더라도 기본 이미지 합성 템플릿(가사와 프롬프트를 텍스트로 오버레이한 정적 배경 이미지)을 기본 제공하여 차단 방어 |
